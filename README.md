# SleepLog - Журнал сна и биоритмов

Консольное приложение для учёта параметров сна, анализа качества отдыха и получения персональных инсайтов.

## Возможности

- 📝 Ежедневный учёт сна (время отхода/пробуждения, качество, энергия, факторы)
- 📊 Аналитика за неделю/месяц (средние показатели, частые факторы)
- 📈 Текстовая визуализация продолжительности сна
- 💡 Персональные инсайты на основе ваших данных
- 💾 Экспорт/импорт в ZIP (CSV + JSON + копия БД)
- 🔄 Автоматическое резервное копирование базы данных
